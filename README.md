# 3s1t editor

- Open Source
- Browser Based
- Integrated Development Environment (IDE)

## Quick Start

- clone repo
- `npm i --legacy-peer-deps`
- `npm run dev`

## Example Routes

- 3s1t.xyz
  - /{project_id}
    - /edit
      - explorer
      - search
      - source control
      - extensions
    - /view
      - explorer
      - search
      - extensions
- searchParams
  - ?tabTreeId={tabTreeId}
